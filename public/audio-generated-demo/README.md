# Audio Generated Demo

This folder contains generated demo songs that are randomly selected when a user generates a song.

## Usage

When a song is generated, the system will randomly select one of the audio files in this folder to play.

## File Format

- Audio files should be in MP3 format
- Files will be randomly selected when "Generate Demo Song" is clicked
- Each regeneration will select a new random song

## Example Files

You can add your generated songs here, for example:
- `generated-song-1.mp3`
- `generated-song-2.mp3`
- `generated-song-3.mp3`
