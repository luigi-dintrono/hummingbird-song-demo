# Audio Style Clips

This folder contains 10-second audio clips for each audio style.

## Required Files

Each audio style needs a corresponding MP3 file named in lowercase:

- `jazz.mp3`
- `punk.mp3`
- `reggae.mp3`
- `country.mp3`
- `pop.mp3`
- `rock.mp3`
- `blues.mp3`
- `opera.mp3`
- `drums.mp3`
- `soul.mp3`
- `indie.mp3`
- `rap.mp3`
- `latin.mp3`
- `classical.mp3`
- `techno.mp3`

## File Requirements

- Format: MP3
- Duration: ~10 seconds
- Sample rate: 44.1kHz recommended
- Bitrate: 128kbps or higher recommended

## Usage

The audio files are automatically loaded when a user selects an audio style in the Audio Reference section. The audio will auto-play when selected and can be controlled with play/pause/restart buttons.
